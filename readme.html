Home # HansHuang917.github.io 以下記錄我怎麼建好這個靜態網站
建立一個新的Repository
靜態網站的名稱必須是{Owner}.github.io，因為我已經建立過，所以會出現紅框
add a readme打勾，然後建立。
進到剛建好的儲存庫，點[<>Code]，然後複製那段網址。
開啟Github Desktop，然後在Current repository點一下，會出現[Add]按鈕，選Clone repository。
把剛複製的URL貼上來，這裡有顯示儲存的路徑，然後按[Clone]
有個進Local path的快速方法，點[Open in Sublime Text]
在資料夾上按右鍵，選Open Folder，就可以了。
把之前做好的靜態網頁(html, javascript, css)丟進資料夾中，
再回去看Github Desktop，輸入必填資料，就可以把檔案commit&push上去了
回到Repository
發現檔案已上傳成功！
這是有萬年曆&記事本功能的網頁，但要加上RWD，不然手機版的memo list看不到
